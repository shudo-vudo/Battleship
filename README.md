# Компьютерная игра “Морской бой"

Морской бой – одна из самых популярных игр в мире, в которую играют люди разных поколений. Игра создана для двух человек, и она очень проста и любыми способами возможно играть в неё. Для игры достаточно расставить корабли на поле размером 10 на 10 и угадывать каждый ход клетки противника, на которых стоит корабль.

В процессе игры пользователь может:  

    ●	Управлять меню игры;  
    ●	Расставлять до начала игры корабли на своем поле или выбрать случайную расстановку кораблей;  
    ●	Выбирать клетку для бомбёжки поля противника;  
    ●	Выходить из игры, начинать заново игру.  
    
Сама игра умеет:  

    ●	Во время игры сохранять данные поля и не показывать их противнику;  
    ●	Соблюдать правила расстановки кораблей;  
    ●	В случае игры с компьютером анализировать поле и предыдущие ходы и следующий ход делать на основе проведенного анализа.
